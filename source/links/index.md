# 推荐工具

这里不是链接农场，也不是随手堆一排按钮。

我把自己长期用过、写过、踩过坑的工具统一放在这里：AI订阅、虚拟卡、出海支付、加拿大生活判断，以及后面会继续增加的人生计算器系列。每个入口都尽量配文章、使用场景和避坑说明，方便你先看清楚自己到底需要什么，再决定要不要折腾。

## 为什么要做这个页

很多问题不是缺一个工具，而是缺一个入口。

比如你以为自己缺的是一张虚拟卡，最后发现真正的问题是KYC、风控、卡段和使用场景；你以为自己是在问要不要移民，最后发现真正的问题是职业迁移、家庭承压和心理重启能力。

所以这个页面会持续更新。能用的留下，过时的标注，踩坑的写清楚。
